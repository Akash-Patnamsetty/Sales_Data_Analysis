Sales Data Analysis Using Python

Project Overview
This project focuses on performing **Exploratory Data Analysis (EDA)** on a retail sales dataset to uncover meaningful business insights.  
Using Python and popular data analysis libraries, the project analyzes sales performance across product categories, regions, and time.

---

Objectives
- Understand overall sales and profit performance
- Identify top-performing product categories
- Analyze regional profitability
- Study monthly sales trends
- Generate actionable business insights using data

---

Tools & Technologies
- **Python**
- **Pandas**
- **NumPy**
- **Matplotlib**
- **Jupyter Notebook**

---

Dataset Description
- Total records: **1,000**
- The dataset contains the following key attributes:
  - Product ID
  - Sale Date
  - Sales Representative
  - Region
  - Product Category
  - Sales Amount
  - Quantity Sold
  - Unit Cost & Unit Price
  - Discount
  - Customer Type
  - Payment Method
  - Sales Channel

---

Data Analysis Steps
1. Loaded and inspected the dataset
2. Converted date columns to proper datetime format
3. Checked for missing values and duplicates
4. Performed feature engineering by creating a **Profit** column
5. Conducted exploratory data analysis using grouping and aggregations
6. Visualized trends using line and bar charts

---

Key Analysis Performed
- Total sales and total profit calculation
- Sales analysis by product category
- Profit analysis by region
- Monthly sales trend analysis
- Comparative analysis using visualizations

---

Key Insights
- **Clothing** is the highest revenue-generating product category
- **North region** contributes the highest overall profit
- Sales show noticeable **seasonal trends**, with higher performance toward year-end
- Profitability varies significantly across regions and categories

---

Visualizations
- Monthly Sales Trend (Line Chart)
- Sales by Product Category (Bar Chart)
- Regional Profit Analysis
